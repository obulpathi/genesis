# Maven

* mvn -B archetype:generate \
  -DarchetypeGroupId=org.apache.maven.archetypes \
  -DgroupId=app \
  -DartifactId=app

* mvn compile
* mvn test
* mvn package
* mvn install (~/.m2/repository)
* mvn clean
* mvn site (generate docs)
* mvn idea:idea 9Generate IntelliJ descriptior for project)
